서비스 이용방법

1. 회원가입

서비스 이용을 위해
회원가입이 필요합니다.

이미 회원 등록을 마치신 분은
로그인을 해주세요.


2. 사진 업로드

전송할 사진을 등록해주세요.

업로드한 사진 중, 랜덤으로 한 장을 선택하여 보내드립니다.

3. 정보 입력

발신인, 수신인의 정보(이름, 전화번호)와 함께 전송할 메시지를 입력해주세요.

발신 기간은 제한이 없으며, 입력한 기간 안에 랜덤으로 전송됩니다.

예를 들어, 기간을 30으로 입력하시면,
랜덤으로 기간을 선택하여(ex 14) 전송 예약을 하는 날로부터 14일 후에 메시지가 전송됩니다.


4. 문자 전송

입력하신 정보를 바탕으로
문자메시지 전송이 예약됩니다.
